# HTML Calculator App

Это простой и удобный веб-калькулятор, который работает прямо в браузере. Проект создан в рамках производственной практики на основе утвержденного каталога пет-проектов. Калькулятор поддерживает базовые вычисления, умеет работать с десятичными дробями, выполнять цепочки операций и правильно сбрасывать историю ввода.

## 🛠 Стек технологий
* **Frontend:** HTML5, CSS3 (адаптивная сетка на CSS Grid Layout)
* **Логика приложения:** Чистый JavaScript (ES6+)

## 🚀 Как запустить проект локально

Вам не нужно ничего скачивать или устанавливать на компьютер, кроме самого кода.

1. **Клонируйте этот репозиторий:**
   ```bash
   git clone https://github.com/cH1PandDal3/HTML-Calculator-JS-CSS.git
   ```
2. **Откройте папку с проектом:**
   ```bash
   cd <HTML-Calculator-JS-CSS>
   ```
3. **Запустите калькулятор:**
   Просто найдите в папке файл `index.html` и дважды кликните по нему. Он откроется в любом вашем браузере.
